 How to use
-----------------
1. This Ultimate Resource Bundle contains ALL premium SPUM addon packs plus additional game resources.
2. IMPORTANT: This pack does NOT include SPUM Manager (Character Creation Tool).
3. To create custom units, you need the base SPUM asset from:
   https://assetstore.unity.com/packages/slug/188715
4. This bundle works with SPUM version 1.8.2 or higher.
5. Addon sprites are located in Assets/SPUM/Resources/Addons/
6. Additional resources (UI, Effects, Maps) are in Assets/SPUM/Ultimate Resource Bundle/
7. You can use the included prefab units standalone without SPUM Manager.
   (But you cannot edit or modify units without SPUM Manager)
   (We recommend using with base SPUM for full functionality)
8. Sample scene demonstrates basic unit management and animation control.
9. Retro UI scenes showcase the included UI framework components.
